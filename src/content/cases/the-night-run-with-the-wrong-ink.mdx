印刷厂的夜班，机器声盖过一切。

3 号机所在的车间本该是熄着的。可巡查员卞澄走到那条侧通道时，脚步停住了——机器在转。

没有人看见接下来发生了什么。等到交班，工人发现卞澄倒在墨槽旁，地上散着几张还没干透的样张，金色的底纹在灯下泛光。

探员捏起一张样张，对着光看那层金墨。

「这颜色，」他说，「这个月厂里没进过。」

## 七、揭晓真相

<RevealAnswer>

### 真相

凶手是夜班领班晁岭。

晁岭利用夜班权限，私自调用 3 号机偷印某种有价票证的防伪活件牟利（银行不明进账即来源）。当晚巡查员卞澄发现 3 号机在运转、却不在生产单上，录下语音备忘准备去质问。晁岭为灭口，在侧通道用墨刀将其杀害，再谎称 3 号机整晚停机检修。

但机器骗不了人：那批金墨不在本月入库记录里、是计划外活件用的；3 号机在"检修"时段计数仍涨约 2000 印、墨辊处于高温运转——证明机器一直在高速印刷，"停机检修"是谎言。

### 关键矛盾

晁岭用"停机检修"否认 3 号机当晚开工，但计数器、墨辊温度、来历不明的防伪金墨样张三者共同证明机器在偷印计划外活件；卞澄的语音备忘正是撞破这件事的瞬间，构成杀人动机。

### 伏笔解析

1. 金墨不在入库记录——计划外活件，私印的物证。
2. 「检修」时段计数涨两千印——机器没停。
3. 墨辊高温运转——印刷在进行，与"检修"矛盾。
4. 22:52 语音备忘——死者撞破偷印，招致灭口。
5. 防伪版纹 + 不明进账——封闭动机链（私印有价票证牟利）。
6. 墨刀指纹与擦拭血迹——凶器与晁岭绑定。
7. 尚牧已下班、邬桑有人证——两名红鲱鱼排除。

真相从不明说。它只能被推断。
